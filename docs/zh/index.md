## IMGPrompt 是什么

[IMGPrompt](https://prompt.newzone.top/app/zh) 是一个面向非英语母语创作者的 AI 图像/视频提示词编辑器。我们收录并人工整理了 **5000+ 英文提示词**，分布在 16 个主类和数百个子类下，每个都配有母语翻译和分类。你在母语界面里点击组合，直接产出标准英文提示词，适配 **Nano Banana、GPT-Image-2、Midjourney、FLUX、Seedance、Veo、Kling** 等主流图像与视频模型（也兼容 Stable Diffusion、DALL·E 等语法）。

## 为什么选择 IMGPrompt

- **母语操作，英文输出**：每个提示词都附母语解释，无障碍浏览和选用。
- **悬停 / 长按即预览**：桌面悬停或触屏长按 0.5 秒即弹出示例图，点击预览图可一键放大、缩放、旋转、下载。
- **双模可切换**：多彩色块 / 单色紧凑两种标签风格一键切换，偏好自动记住。
- **18 种语言界面**：简/繁中、英、日、韩、西、法、德、葡、俄、阿、印地、孟、印尼、越、泰、土、意，整库同步本地化。
- **链接分享浏览状态**：URL 参数保存当前主类/子类，队友打开网址看到同一视图。
- **完全本地运行**：无需登录、无数据上传、无后端；Windows / macOS (Intel & Apple Silicon) / Linux 桌面客户端离线可用。
