# 界面说明

IMGPrompt 界面分为 **左侧选词区** 和 **右侧结果区** 两部分，从上到下按"**1 选对象 → 2 选属性 → 3 选标签**"三个步骤编号引导，流程清晰。

![界面说明](https://img.newzone.top/2024-10-23-18-31-38.png?imageMogr2/format/webp)

## 1. 选择对象（主分类）

顶部 **① 选择对象** 区域列出 16 个主分类，例如 **人物 / 环境 / 摄影 / 艺术风格 / 光影** 等。点击任一分类切换当前主题。

## 2. 选择属性（子分类）

**② 选择属性** 会根据当前主分类动态切换出对应子类，例如"人物"下会有 **基础 / 表情 / 动作 / 肖像 / 服饰** 等。点击子分类进一步缩小范围。

## 3. 浏览并插入标签

**③ 选择标签** 区域展示当前分类下的全部提示词标签，双语显示（英文 displayName + 母语 langName）。

### 点击插入 / 移除

点击任意标签即自动添加到右侧提示词框；再次点击已选标签即可移除。系统自动去重，重复不计。

### 悬停即预览，点击可放大 🖼️

带有示例图的标签，鼠标悬停时会自动弹出预览浮窗；**点击浮窗内的预览图**即可打开 lightbox：

- **放大 / 缩小**：滚轮或按钮缩放，看清细节。
- **旋转**：45°/90° 旋转，多角度查看。
- **下载**：一键保存到本地参考图库。
- **键盘操作**：`Esc` 关闭，左右键切换（如适用）。

关闭 lightbox 后，当前浏览位置和已选标签完全保留，不打断挑选流程。

### 多彩 vs 单色：一键切换 🎨

右上角的模式切换器提供两种视觉风格：

- **多彩模式**：按标签类别用不同色块区分，视觉层次强，适合浏览和灵感激发。
- **单色模式**：信息密度更高，适合快速扫读和对比已选。

切换偏好会自动保存到浏览器本地，下次打开沿用。

## 4. 右侧结果区 —— 编辑与推荐

### 直接编辑提示词

右侧文本框支持直接输入自定义内容。**用英文逗号 `,` 分隔**不同提示词，系统自动完成标签化、去重、匹配。

### 智能推荐

根据你最后输入的词，自动推荐 **10 条相关标签**（中英文皆可）。点击推荐即可插入。若最后一个词匹配到词库，会以绿色打勾标签形式显示。

![](https://img.newzone.top/2024-12-23-22-13-55.png?imageMogr2/format/webp)

### 实用按钮

| 功能 | 说明 |
| --- | --- |
| **肖像光线** | 一键追加常用人像灯光模板（如 soft light、rim light 等）。 |
| **润色增强** | 追加画质、细节、清晰度等通用润色词。 |
| **否定提示** | 复制常用负面提示词（如 blurry、bad anatomy 等）。 |
| **翻译** | 将任意语言（1.5s 防抖自动触发）转为英文，多服务降级确保成功率。 |
| **随机换色** | 随机替换提示词中的颜色词，激发灵感。 |
| **复制结果** | 一键复制到剪贴板，粘贴到任意 AI 工具。 |
| **清空** | 清除所有已选标签与输入内容，快速开始新会话。 |

### 字符计数器 📊

实时显示提示词长度，建议 **≤ 380 字符**，利于模型稳定复现。

## 5. 链接分享：把当前视图直接发给别人 🔗

浏览时 URL 会自动同步当前分类状态：

```
https://prompt.newzone.top/app/zh?object=人物&attribute=肖像
```

把这个网址发给队友，他们打开就看到同一个分类视图。非常适合：

- 团队协作时指向同一组可选标签
- 教程 / 文章里引用特定分类
- 收藏自己常用的分类组合

## 6. 主题与语言

- **明暗主题**：跟随系统、或手动切换（顶部导航栏）。
- **18 种语言**：整个界面 + 标签库 + 分类名称全量本地化；在导航栏切换语言即可。

## 7. 其他贴心细节

- **关闭页面不丢失**：已选标签、主题偏好、模式偏好均保存在本地。
- **完全本地运行**：无登录、无上传、无后端 —— 浏览器完成全部运算。
- **键盘无障碍**：所有标签支持 Tab 聚焦 + `Enter` / `Space` 触发，适配读屏器。
