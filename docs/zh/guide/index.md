# 介绍

在 AI 图像生成的创作中，准确的提示词是高质量作品的关键。对非英语母语者来说，最难的是找到恰当的英文表达。**IMGPrompt** 以“母语浏览，英文输出”为理念，帮你无障碍组合出适用于 Nano Banana、Midjourney、FLUX、Stable Diffusion、DALL·E 乃至 Sora 的高质量提示词。

我们提供 **5000+ 精选提示词**、清晰直观的界面，以及每个标签的母语翻译，让你用点击式操作在几秒内拼好完美的英文提示词。

如果你对 AI 文生图完全不了解，可先查看攻略：[零基础入门 Stable Diffusion - 无需显卡把 AI 绘画引擎搬进家用电脑](https://newzone.top/posts/2022-09-05-stable_diffusion_ai_painting.html)。

## 为什么选择 IMGPrompt？

- 母语操作，输出流畅英文，适配主流模型。
- 5000+ 经过整理的标签，涵盖风格、灯光、构图、艺术家等。
- 可视化点击组合，保持创作节奏。

## 如何使用

1. 在母语界面浏览提示词标签。
2. 点击添加风格、灯光、主体与细节。
3. 复制生成的英文提示词到任意 AI 工具。

## 特色功能

### 🏆 5000+ 本地-英语对照库

提示词库配有母语解释，让你确认含义的同时输出标准英文。

### 🧩 智能模板与工具

- 一键润色：如人像灯光、高质量优化。
- 提示词去重与清理，让结果更简洁。
- 随机色彩与风格灵感，帮你打破创作瓶颈。

### 🤖 智能推荐与翻译

- 输入 17 种语言的关键词即可获得相关标签推荐。
- 内置翻译把任意文本转成英文，保持跨模型兼容。

![翻译展示](https://img.newzone.top/gif/imgprompt-translate.webp)

### 📂 开放可自定义数据库

所有提示词数据存放在开放的 JSON（`src/app/data/prompt/*.json`），可直接编辑，加入你的习惯用法。

### 🖥️ 跨平台客户端

偏好桌面端？前往 [GitHub Releases](https://github.com/rockbenben/img-prompt/releases) 下载 Windows、macOS、Linux 版本。

### 🌍 多语言支持

界面与提示词数据库支持 17 种语言，内置翻译工具也能处理自定义输入。

### ⚡ 工作流小助手

- 逗号标签化并自动去重，按类别展示便于查找。
- 点击标签即可快速增删，编辑体验流畅。
- 字符统计建议将提示词控制在约 380 字符内。

IMGPrompt 的提示词翻译功能采用有道翻译、MyMemory 和 Google Translate API，欢迎反馈与报错。

需要批量处理数据？试试 [IMGPrompt 数据转换器](https://tools.newzone.top/data-parser/img-prompt) 轻松导入和转换提示词数据。
