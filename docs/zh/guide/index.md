# 介绍

在 AI 图像/视频生成创作中，准确的提示词是高质量作品的关键。对非英语母语者来说，最难的是找到恰当的英文表达。**IMGPrompt** 以"母语浏览，英文输出"为理念，帮你无障碍组合出适用于 Nano Banana、GPT-Image-2、Midjourney、FLUX、Seedance、Veo、Kling 等主流模型的高质量提示词（也兼容 Stable Diffusion、DALL·E 等语法）。

我们提供 **5000+ 精选提示词**、清晰直观的界面、每个标签的母语翻译，以及带图条目的即时预览，让你用点击式操作在几秒内拼好完美的英文提示词。

## 为什么选择 IMGPrompt

- 母语操作，输出流畅英文，适配主流图像/视频模型。
- 5000+ 整理好的标签，16 大类覆盖风格、灯光、构图、镜头、艺术家等。
- 可视化点击组合，桌面悬停 / 触屏长按即预览示例图，保持创作节奏。

## 如何使用

1. 在母语界面浏览 16 个主分类。
2. 点击主类 → 子类，快速缩小范围。
3. 点击标签加入提示词框；桌面悬停或触屏长按 0.5 秒查看示例图，再点击缩略图放大。
4. 复制生成的英文提示词到任意 AI 工具。

## 特色功能

### 🏆 5000+ 母语-英语对照库

16 大类、数百子类、每个标签均配母语解释，让你确认含义的同时输出标准英文。

### 🖼️ 悬停 / 长按预览，点击可放大

带图条目在桌面端鼠标悬停、在触屏设备**长按 0.5 秒**会弹出示例图浮窗（含母语全名、缩略图、说明），点击缩略图即可进入 lightbox：支持放大、缩小、旋转、下载，帮助你"所见即所得"地理解每个提示词的实际效果。关闭 lightbox 后浏览状态完全保留，不打断挑选流程。

> 触屏端单击仍是「加入 / 移除标签」，只有长按才唤起预览，互不干扰。

### 🎨 双模切换：多彩 vs 单色

- **多彩模式**：按分类用不同色块区分标签，视觉层次强，适合浏览和灵感激发。
- **单色模式**：信息密度高，适合快速扫读和比对已选。
- 右上角切换开关，偏好自动保存到本地，下次打开直接沿用。

### 🔗 链接分享浏览状态

URL 参数 `?object=主类&attribute=子类` 会自动跟随当前浏览位置变化。把网址发给队友，他们打开就看到同一个分类视图，非常适合团队协作和教程示范。

### 🤖 智能推荐与翻译

- 18 种语言输入关键词即可获得 10 条相关标签推荐（中英文皆可）。
- 内置翻译把任意文本转为英文，多服务降级（Google → Youdao → MyMemory）+ 竞态保护，保证跨模型兼容。
- 1.5s 防抖避免频繁请求。

### 🧩 一键模板与清理

- 快速插入：肖像光线、高质量润色、负面提示等常用模板。
- 逗号标签化：输入逗号即自动去重、分组、标签化，不手动清理。
- 随机换色：一键替换提示词中的颜色词，打破创作瓶颈。

### 📊 字符计数与模型友好

实时显示当前提示词字符数，建议控制在 **380 字符以内**，提升模型复现稳定性。

### 🌓 明暗双主题

完整支持浅色 / 深色 / 跟随系统三档，antd 6 token 化调色板，夜间浏览不刺眼。

### 🔒 完全本地运行

所有操作在浏览器端完成，无需登录、无数据上传、无后端数据库。切换页签也不丢状态。

### 📂 开放可自定义数据库

所有提示词数据存储为开放的 JSON（`src/app/data/prompt/*.json`），可直接编辑，也可用 [IMGPrompt 数据转换器](https://tools.newzone.top/data-parser/img-prompt) 批量导入自己的关键词。

### 🖥️ 跨平台客户端

偏好桌面端？前往 [GitHub Releases](https://github.com/rockbenben/img-prompt/releases) 下载 Windows、macOS（Intel + Apple Silicon）、Linux 版本。

### 🌍 18 种语言支持

界面与提示词数据库同步支持：简/繁中、英、日、韩、西、法、德、葡、俄、阿、印地、孟、印尼、越、泰、土、意。
